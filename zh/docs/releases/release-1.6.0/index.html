Cloudberry Database v1.6.0 发版说明

版本号：v1.6.0

Cloudberry Database v1.6.0 是一个小版本，包含一系列提升改进、变更和 bug 修复。

快速试用：v1.6.0

完整的变更日志：https://github.com/cloudberrydb/cloudberrydb/compare/1.5.4...1.6.0

提升与变更​
